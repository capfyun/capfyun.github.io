    屏蔽方式主要有两种：整个网站无法访问（域名劫持：使域名无法转换成正确的IP地址，无法找到网站； IP封锁：使你找不到目标网站的主机地址），包含特定内容的网页无法访问，即关键字过滤，不过最近听说百度能查小草的免代地址了
      
      代理服务器主要的类型包括HTTP代理和SOCKS代理。
      HTTP代理：最简单的一种代理形式，能够代理客户机的HTTP访问，上网浏览网页使用的都是HTTP协议，通常的HTTP代理端口为80、3128或8080端口。
      
      SOCKS代理：SOCKS代理与HTTP等其他类型的代理不同，它只是简单地传递数据包，而并不关心是何种应用协议，既可以是HTTP协议，也可以是FTP协议，或者其他任何协议，所以SOCKS代理服务器比其他类型的代理服务器速度要快得多。目前SOCKS5是最常用的一种SOCKS代理。

                            修改host篇

    注：用于打开被 GFW 屏蔽了域名，但未屏蔽 IP 地址的网站。

     

    Hosts 文件位置：
    在Windows 2000/XP/Vista/Win7 系统中位于：系统安装路径\System32\Drivers\Etc 文件夹内
    Android 手机的 hosts 文件在system/etc目录下，注意替换完成后需修改文件权限。
    iPhone OS：/etc
    iPad OS：/private/etc

    例如，Windows XP 安装在C:\WINDOWS，那么 Hosts 文件就在C:\WINDOWS\system32\drivers\etc中。
    * 也可以用windows自带的查找功能搜索找到 hosts 文件。

    取得网站 IP 地址的方法：
    1.在您的电脑上请按如下步骤操作：开始-->运行-->输入cmd-->ping 域名-->回车查看结果
    2.显示结果类似 Reply from 220.181.31.183: bytes=32 time=79ms TTL=53  中间的  220.181.31.183 就是域名的IP地址。

    修改方法：
    *　该文件是一个纯文本的文件，用记事本就能打开和编辑。
    *  当网站更换ip 地址后，Hosts 文件内的相应内容也应被手工更新，否则会无法访问网站。
    *  每段只能包括一个IP地址和一个与之有映射关系的主机名。
    *  IP地址要放在每段的最前面，映射的Host name(主机名)在IP后面，中间用空格分隔。对于这段的映射说明，用“#”分割后用文字说明。

    查找域名的IP地址的网站：
    http://just-ping.com/
    http://tools.pingdom.com/ping/
    http://ping.eu/ping

    两个快速修改Host的程序：Hosts File Patch ， onekeyhost (一键修改hosts文件)。

    https://sites.google.com/site/hostsfilepatch/
